# Responsive Landing Page

## Overview

This project is a responsive landing page developed as part of the SkillCraft Technology Web Development Internship. The application includes an interactive navigation menu with dynamic styling changes on scrolling and hover actions.

## Features

- Responsive user interface
- Interactive navigation menu
- Fixed navigation bar
- Hover effects
- Scroll-based styling changes
- User-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

├── index.html

├── script.js

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
